# 🛒 Superstore Sales Analysis

![Dashboard](Superstore%20Sales%20Dashboard.png)

## 📌 Project Overview
Complete end-to-end sales data analysis of the Sample Superstore 
dataset using Python, Pandas, and Power BI. This project covers 
data loading, exploratory data analysis (EDA), visualization, 
and an interactive Power BI dashboard.

## 🛠️ Tools Used
- Python, Pandas, Matplotlib, Seaborn
- Power BI Desktop (DAX Measures, KPI Cards, Slicers)
- Jupyter Notebook

## 📊 Analysis Done
- Region-wise Total Sales
- Category-wise Sales Distribution
- Top 10 Products by Sales
- Monthly Sales Trend
- Segment-wise Sales Distribution
- Ship Mode Usage
- Profit Analysis by Category
- Discount vs Sales Impact
- Quantity Sold by Category

## 📈 Power BI Dashboard
Interactive dashboard with:
- 6 KPI Cards (Total Sales, Total Profit, Total Orders, 
  Total Customers, Profit Margin %, Total Quantity)
- 4 Slicers (Year, Region, Category, Segment)
- 6 Charts (Sales Trend, Region, Category, 
  Segment, Sub-Category, Ship Mode)

## 🔍 Key Findings
- Technology leads in both Sales (0.84M) and Profit (1.45L)
- Consumer segment dominates with 50.8% of total sales
- West region generates highest sales
- Office Supplies has highest quantity sold (22,906 units)
- Standard Class is most preferred ship mode
- Sales show consistent YoY growth from 2014 to 2017
- Furniture has lowest profit margin despite high sales

## 📁 Dataset
- Source: Sample Superstore Dataset (Kaggle)
- Records: 9,994 rows | 21 columns
